# Application Note

This repository is a practical AI-native workflow artifact for a VC Analyst / Sourcing & Ops role.

It demonstrates how I would create operating leverage for a VC fund by combining:

- inbound founder screening
- company and website analysis
- structured fund-fit assessment
- pipeline hygiene
- reusable notes and reporting formats
- human-in-the-loop approval for sensitive actions

The goal is not to replace investment judgment. The goal is to reduce manual work, make messy inputs easier to evaluate, and help a partner move faster without dropping context or next steps.
